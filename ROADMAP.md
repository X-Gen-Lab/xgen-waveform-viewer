# xgen-waveform-viewer 开发路线图

本文档概述了 xgen-waveform-viewer 的未来发展方向和计划实现的功能。

## 已完成 ✅

### V2.0 - 基础功能
- [x] 实时串口 ADC 数据采集
- [x] CRC-16 帧校验和自动重同步
- [x] 实时波形显示（pyqtgraph）
- [x] 数据录制（BIN v2 / CSV）
- [x] X/Y 轴灵活控制
- [x] 环形缓冲区管理
- [x] GitHub Actions 自动发布

### V2.1 - 用户体验增强 ✅
- [x] **配置持久化** - 自动保存/恢复用户设置
- [x] **键盘快捷键** - 快速操作支持
- [x] **主题切换** - 暗色/亮色主题
- [x] **自动重连** - 串口断连自动恢复
- [x] **菜单栏** - 更好的功能组织
- [x] **关于对话框** - 版本和许可信息

## 计划中 📋

### V2.2 - 数据分析工具 ✅ (Completed)
- [x] **测量工具**
  - [x] 标尺（Ruler）：测量时间间隔和幅值差
  - [x] 峰值检测与标注
  - [x] 频率/周期自动计算
  - [x] RMS、平均值、最大最小值显示
- [x] **触发功能**
  - [x] 边沿触发（上升/下降）
  - [x] 阈值触发
  - [x] 单次触发模式
- [x] **录制增强**
  - [x] 自动分段录制（按时间或大小）
  - [x] 支持暂停/恢复录制
  - [x] 录制预览（时长和文件大小）

### V2.3 - 性能与稳定性 ✅ (Completed)
- [x] **性能优化**
  - [x] 分段 min/max 降采样算法
  - [x] 高采样率（>10kHz）渲染优化
  - [x] 帧率限制选项
  - [x] 大数据量（>10M）内存优化
- [x] **鲁棒性提升**
  - [x] 增强 CRC 错误恢复策略
  - [x] 异常日志记录系统
  - [x] 丢帧可视化统计图表
  - [x] 录制数据完整性检查

### V2.4 - 数据管理与回放 ✅ (Completed)
- [x] **数据回放**
  - [x] 录制文件回放功能
  - [x] 变速播放 (0.1x ~ 10x)
  - [x] 播放进度条
- [x] **导出选项**
  - [x] 导出当前可见区域为图片（PNG/SVG）
  - [x] 导出统计报告（HTML）
  - [x] 支持 MATLAB 格式（.mat）
- [x] **文件格式**
  - [x] HDF5 高效存储格式
  - [x] 波形比较（叠加多个文件）

### V3.0 - 专业化与扩展 (Q4 2025)
- [ ] **多通道支持**
  - [ ] 重构数据模型支持多通道
  - [ ] 通道分组显示
  - [ ] 通道颜色和标签管理
- [ ] **协议扩展**
  - [ ] 自定义帧格式（配置文件）
  - [ ] ASCII 协议支持
  - [ ] TCP/UDP 数据源
- [ ] **固件配置**
  - [ ] 通过上位机配置采样率、帧长
  - [ ] 固件版本检测与兼容性提示
  - [ ] OTA 固件更新支持

### V3.x - 平台与生态 (2026)
- [ ] **跨平台优化**
  - [ ] Linux 串口权限管理完善
  - [ ] AppImage / Snap 打包
  - [ ] macOS 支持与 DMG 安装包
  - [ ] Apple Silicon 适配
- [ ] **插件系统**
  - [ ] 插件架构实现
  - [ ] 插件 API 文档
  - [ ] 插件市场/仓库
- [ ] **自动化与脚本**
  - [ ] Python API（自动化测试）
  - [ ] 宏录制/回放
  - [ ] 命令行批处理模式
- [ ] **文档与社区**
  - [ ] 详细用户手册
  - [ ] 开发者文档

路线图会根据以下因素调整：
- 用户反馈和功能请求
- 技术可行性评估
- 团队资源和时间
- 依赖项更新和兼容性

如果您认为某个功能应该优先实现，请在 Issues 中投票或评论。
