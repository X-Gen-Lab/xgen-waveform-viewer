# V2.4 功能快速使用指南

本指南帮助您快速上手 xgen-waveform-viewer V2.4 的新功能。

## 📼 数据回放

### 基本使用

1. **打开回放面板**
   - 方法 1: 按 `Ctrl+P`
   - 方法 2: 菜单 `File` → `Playback Recording...`

2. **选择回放文件**
   - 点击"打开文件..."按钮
   - 支持 `.bin` 和 `.csv` 格式

3. **调整播放速度**（可选）
   - 从下拉菜单选择速度：0.1x ~ 10x
   - 默认为 1x（原始速度）

4. **开始播放**
   - 点击"▶ 播放"按钮
   - 波形将在主窗口中显示

5. **播放控制**
   - **暂停**: 点击"⏸ 暂停"
   - **恢复**: 点击"▶ 恢复"
   - **停止**: 点击"⏹ 停止"

### 进度显示

回放面板会显示：
- 当前播放时间 / 总时长
- 进度条（可视化）
- 文件信息（格式、采样率、总帧数等）

### 注意事项

- 回放时串口连接会自动禁用
- 停止回放后可重新连接串口
- 所有查看功能（缩放、平移、测量）在回放模式下仍可使用

## 🎨 高级导出功能

### PNG 图片导出

**用途**: 报告、演示文稿、文档配图

**步骤**:
1. 调整波形显示到理想视图
2. 菜单 `File` → `Export As` → `Export as PNG...`
3. 选择保存位置
4. 导出分辨率：1920x1080

**提示**: 导出前可以：
- 调整 X/Y 轴范围
- 启用测量工具（标尺、峰值标记）
- 这些都会包含在导出图片中

---

### SVG 矢量图导出

**用途**: 论文、出版物、可缩放图形

**步骤**:
1. 调整波形显示
2. 菜单 `File` → `Export As` → `Export as SVG...`
3. 选择保存位置

**优势**:
- 矢量图形，无损缩放
- 文件体积小
- 适合印刷品质要求

---

### MATLAB 格式导出

**用途**: 科学计算、数据分析、MATLAB/Octave 处理

**前置条件**:
```bash
pip install scipy
```

**步骤**:
1. 菜单 `File` → `Export As` → `Export as MATLAB (.mat)...`
2. 选择保存位置
3. 文件包含：`samples`（数据）、`time`（时间）、`sample_rate_hz`（采样率）

**在 MATLAB 中使用**:
```matlab
% 加载数据
data = load('waveform.mat');

% 绘制波形
plot(data.time, data.samples);
xlabel('Time (s)');
ylabel('ADC Value');
title(sprintf('Sample Rate: %d Hz', data.sample_rate_hz));

% 频谱分析
Fs = data.sample_rate_hz;
N = length(data.samples);
f = (0:N-1)*(Fs/N);
Y = fft(data.samples);
plot(f, abs(Y));
xlabel('Frequency (Hz)');
ylabel('Magnitude');
```

### HDF5 格式导出

**用途**: 大数据存储、高效压缩、科学数据标准格式

**前置条件**:
```bash
pip install h5py
```

**步骤**:
1. 菜单 `File` → `Export As` → `Export as HDF5 (.h5)...`
2. 选择保存位置
3. 自动启用 gzip 压缩（通常节省 50-70% 空间）

**在 Python 中使用**:
```python
import h5py
import numpy as np
import matplotlib.pyplot as plt

# 加载数据
with h5py.File('waveform.h5', 'r') as f:
    samples = f['samples'][:]
    sample_rate = f.attrs['sample_rate_hz']
    duration = f.attrs['duration_s']
    
    # 生成时间数组
    time = np.arange(len(samples)) / sample_rate
    
    # 绘制
    plt.plot(time, samples)
    plt.xlabel('Time (s)')
    plt.ylabel('ADC Value')
    plt.title(f'Duration: {duration:.2f}s')
    plt.show()
```

**优势**:
- 高压缩率（50-70% 空间节省）
- 支持超大数据集
- 保留完整元数据
- 科学界广泛使用

## 📊 波形比较（编程接口）

### 使用场景

- 对比不同测试条件下的波形
- 验证信号一致性
- 分析波形差异

### 示例代码

```python
from xgen_waveform_viewer.exporter import WaveformComparator
import numpy as np

# 假设有两个波形数据
waveform1 = np.load('test1.npy')
waveform2 = np.load('test2.npy')

# 比较波形
comparison = WaveformComparator.compare_waveforms(
    waveform1,
    waveform2,
    sample_rate_hz=10000
)

# 查看结果
print("波形 1:")
print(f"  平均值: {comparison['waveform1']['mean']:.2f}")
print(f"  标准差: {comparison['waveform1']['std']:.2f}")

print("\n波形 2:")
print(f"  平均值: {comparison['waveform2']['mean']:.2f}")
print(f"  标准差: {comparison['waveform2']['std']:.2f}")

print("\n差异:")
print(f"  平均值差: {comparison['difference']['mean_diff']:.2f}")
print(f"  相关性: {comparison['difference']['correlation']:.4f}")

if 'sample_mse' in comparison['difference']:
    print(f"  均方误差: {comparison['difference']['sample_mse']:.2f}")
```

## 🚀 工作流示例

### 完整的测试流程

```
1. 实时采集
   ↓ 连接串口，开始采集
   ↓ 使用触发功能捕获特定事件
   ↓ 开始录制

2. 数据录制
   ↓ 录制为 BIN 格式
   ↓ 必要时使用暂停/恢复
   ↓ 停止录制

3. 数据回放
   ↓ Ctrl+P 打开回放面板
   ↓ 选择录制文件
   ↓ 调整速度，仔细查看
   ↓ 使用测量工具分析

4. 数据导出
   ↓ 导出 PNG 图片（用于报告）
   ↓ 导出 MATLAB 格式（深度分析）
   ↓ 生成 HTML 报告（存档）

5. 波形比较
   ↓ 比较多次测试结果
   ↓ 验证一致性
```

## ⚙️ 配置建议

### 完整功能安装

```bash
# 基础安装
pip install -r requirements.txt

# 安装可选依赖（推荐）
pip install scipy h5py

# 或一次性安装
pip install -e ".[full]"
```

### 性能优化

对于大文件回放：
1. 增加缓冲区大小：`Buf` 设置为 20+ Mpts
2. 降低 FPS 限制：减少渲染负载
3. 使用 HDF5 格式：最高效的存储

## 🔧 故障排除

### 问题：MATLAB/HDF5 导出不可用

**原因**: 未安装可选依赖

**解决**:
```bash
pip install scipy h5py
```

### 问题：回放文件加载失败

**检查**:
- 文件是否完整（未损坏）
- 格式是否支持（BIN 或 CSV）
- 文件头是否包含必要信息

### 问题：PNG/SVG 导出为空

**原因**: 当前缓冲区无数据

**解决**:
- 先采集或回放数据
- 确保波形显示正常
